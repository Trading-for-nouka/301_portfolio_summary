# 📊 301_portfolio_summary — 週刊ポートフォリオサマリー

毎週土曜日に 201〜204 の全戦略の保有ポジションを集計し、
銘柄ごとの損益・保有日数をまとめて Discord に通知します。

## 集計対象

| リポジトリ | 戦略 |
|---|---|
| 201_breakout | ブレイクアウト |
| 202_dip | 押し目買い |
| 203_nr4 | NR4 |
| 204_rebound | リバウンド |

## スケジュール

毎週土曜日 09:00 JST（`cron: '0 0 * * 6'`）

## Secrets

| 名前 | 内容 |
|---|---|
| `DISCORD_WEBHOOK` | Discord の Webhook URL |
| `PAT_TOKEN` | 201〜204 の positions.json 読み取り用 |
